## 문제 설명
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 해결 과정
인덱스가 0 부터 시작해서 +1 해야함을 기억하자
### 접근 방법
for문을 사용 했지만 range에서 n+1를 안해서 계속 틀림
## 코드
```python
n = int(input())
su = 0
for i in range(1,n+1):
    su += i

print(su)
